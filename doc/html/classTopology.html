Detailed Description

Topological information (bonds graph) for one molecule. It maps to one compound in Molmodel thus it is derived from Molmodel Compound class. It does the following things:

  • loads information from input files such as Amber input prmtop / inpcrd
  • adds parameters to a DuMM force field which belongs to the World class because one DuMM class should be used for multiple molecules
  • contructs the graph based on a list of bSpecificAtom objects each of which already contains bonding information from the input files
  • defines the rigid bodies based on imput files provided by the users. Contains a list of atoms bAtomList which consists of bSpecificAtom objects
