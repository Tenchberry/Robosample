Detailed Description

gMolmodel Specific Atom Type Class. This incorporates additional Amber forcefield data.

Constructor & Destructor Documentation

bSpecificAtom::~bSpecificAtom ( )

We do not deallocate bAtomType here. We leave this task to the Topology class that owns this atom in order to allow the number of atoms connected to this to change (e.g. semi-grand canonical ensemble).
